 HTTP隧道
HTTP是基于文本的通过浏览器检索网页的协议。 大多数情况下在代理服务器的后面，通过LAN接入互联网。 在IE的Connection Option中， 给出LAN的设置。 这个代理服务器运行着基于文本的协议， 从它那里可以得到外界的网络HTTP相关的数据。是的， 用HTTP通过它上面的小的望孔可以连接到外部世界， 并用二进制协议得到想要的数据， 或者甚至是协议。 它通过HTTP。
